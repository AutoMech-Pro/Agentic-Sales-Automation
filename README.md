# Agentic-Sales-Automation
 

**Source Material Analysis:**
* **Core Subject:** A technical automation workflow connecting n8n, LinkedIn Data, and Claude 3.7 (via OpenRouter) to write personalized sales emails.
* **Key Data Points:**
    * **Workflow Steps:** 4 distinct stages (Trigger -> Mock/Real Data -> AI Generation -> Output).
    * **Tech Stack:** n8n, OpenRouter, Anthropic Claude 3.7, Proxycurl (optional).
    * **Input Variables:** Name, Role, Company, About Section, Recent Post.
    * **Output Goals:** Short (<100 words), Punchy, Personalized Hook, Soft CTA.
    * **Efficiency:** Automating a manual research process.

**Plan Narrative & Structure:**
1.  **Header & Vision:** Introduce the concept of "Agentic Sales Automation."
2.  **The Tech Ecosystem:** Visualize the integration of tools (Goal: Organize).
3.  **Efficiency Impact:** Quantify the time saved vs. manual research (Goal: Compare).
4.  **Workflow Architecture:** A visual flowchart of the 4-step process defined in the README (Goal: Process).
5.  **The AI Advantage:** A Radar chart comparing "Generic Templates" vs. "Claude 3.7 Generated" emails across quality metrics (Goal: Compare).
6.  **Data Processing Dimensions:** A 3D visualization showing how the AI triangulates data points (Role, Industry, Activity) to find a "Hook" (Goal: Relationships).
7.  **Prerequisites & Configuration:** A clear checklist for implementation (Goal: Inform).

**Visualizations Selected (NO SVG, NO Mermaid):**
1.  **Process Flow:** Structured HTML/CSS Diagram using Tailwind grids and borders. (Goal: Organize/Process).
2.  **Comparison Bar Chart (Chart.js):** "Time Spent per Prospect: Manual vs. Automated". (Goal: Change/Efficiency).
3.  **Quality Radar Chart (Chart.js):** "Email Effectiveness Scoring". (Goal: Compare).
4.  **3D Scatter Plot (Plotly WebGL):** "The Personalization Sweet Spot" - Mapping Role, Recent Activity, and Company data. (Goal: Relationships).

**Palette Selection:**
